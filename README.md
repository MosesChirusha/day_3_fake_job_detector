# Day 3/90 Fake Job Ad Detector

## Problem
Thousands of Africans fall for fake job adverts every year.
Scammers request fees for interviews, uniforms, or registration.
People lose money, time, and personal data.

## Solution
A simple web page where users paste a job advert.
The app scans for red-flag keywords and returns:
- Risk level (Low / Medium / High)
- Alerts about dangerous content
- Security advice
